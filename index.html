GAODING AI MODEL GUIDE
SELECTION LOGIC

选型不是找最强模型，而是匹配任务风险

01

先定任务

图片生成 / 视频生成 / 图像编辑。任务类型决定候选池。

02

再定质量线

草案、运营图、印刷物料、品牌主视觉，对稳定性要求不同。

03

最后控成本

批量探索用低成本，正式交付再切换高稳定或满血模型。

默认策略：先低成本探索方向，再用高质量模型做终稿。

GAODING AI MODEL GUIDE
OPERATING RULES

团队使用规则：避免把豆花在不确定性上

规则 1需求不清时，不直接上高成本模型。
规则 2正式物料看文字、排版、一致性，不只看第一眼画质。
规则 3视频先低规格验证镜头，再升分辨率和音频。
规则 4每次交付记录：模型、消耗、提示词、返工原因。

GAODING AI MODEL GUIDE
TAKEAWAY

结论：用模型矩阵管理创作成本

低成本模型负责探索，高稳定模型负责交付，编辑工具负责压缩返工。

这份明细的价值不只是知道“每次消耗多少豆”，而是建立一套可复用的生产决策：场景匹配、成本分层、质量复核、经验沉淀。
